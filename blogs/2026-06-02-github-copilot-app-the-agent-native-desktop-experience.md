GitHub introduced the Copilot app, a desktop experience designed for managing multiple AI agents working in parallel on development tasks. The platform includes isolated git worktrees for concurrent sessions, canvases for inspecting agent work, cloud and local sandboxes for code execution, and enhanced code review capabilities that scale with agent output.
